# BILLING-APPLICATION

A **full-stack Billing Application** built using **Java Spring Boot** for the backend and **React.js** for the frontend. The system supports **Admin, Customer, and Accountant** roles and provides complete billing and invoice management with user authentication.

## Features

- **Role-based Access**:
  - **Admin**: Full access to manage users, view reports, and monitor the system.
  - **Customer**: Create and view invoices, manage personal details.
  - **Accountant**: Manage invoices, payments, and financial records.
  
- **User Authentication**: Login and registration with email verification.

- **Invoice/Billing Management**:
  - Create, edit, and view invoices.
  - Role-based invoice access.
  
- **Frontend**: React.js SPA for dynamic and responsive user interface.

- **Backend**: Java 21, Spring Boot, Spring MVC, Spring Data JPA, Hibernate, MySQL (or H2 for testing), Spring Security.
